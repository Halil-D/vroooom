# OpenGL 2D Engine

A small game engine written in C/C++ using:

- Win32 API
- OpenGL 3.3
- GLAD
- Custom BMP loader

## Features

- Hardware accelerated rendering
- BMP texture loading
- Delta time
- Gravity and jumping physics
- Fullscreen window

## Controls

A - Move left
D - Move right
Space - Jump
Esc - Quit
